# owspace
owspace是基于MVP+Dagger2+Retrofit2.0+Rxjava的APP

#简介
owspace是一款高仿单读的APP，API接口通过非法手段获取。<br>

#你能学到
* MVP模式
* Retrofit2.0 + rxjava
* Dagger2
* jsoup解析DOM
* sqldelight SqlBrite
* 音视频播放等
* some fucking source code
